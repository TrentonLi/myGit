## 全局命令行工具 myGit，通过 npm 全局安装，在任何项目中调用，带有交互式界面和常用 Git 操作
> 以下步骤完成后,可在任意项目终端执行 ‘mygit’ 命令,进行交互式git操作

### 1.安装依赖
```
npm install
```

### 2.添加执行权限(MAC系统)
```
chmod +x bin/mygit.js
```

### 3.本地全局安装
```
npm link
```
